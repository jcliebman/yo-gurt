Soccer Duel

Download
Soccer Duel, a 1v1 soccer game created by Felix Figueroa. Heavy players tend to win the ball in duels, and they have the best shooting stats. Light players are crazy fast, but they are weak and have poor shooting. Normal players are average, running and shooting well, but they have no special advantage. Ensure that Python 3 is installed on your device and unzip soccergame.zip.
Option 1:
Open README.txt. Follow the instructions to play. The device must use bash. Note that due to security restrictions you may not be allowed to run the file, but you can try to copy paste it into a new file and follow instructions using that file's name instead.
Option 2:
Open the .ttf file and install the font. You can now run game.py any way you wish.

Pong Game with Power Ups

Download
Michael Estevez brings the classic game Pong back to life, but with a twist! There are two modes in this game: single player and multiplayer. Single player allows players to choose between 3 difficulties: Easy, Medium, and Hard. Use the shift keys for the power ups and have fun! Download the file and run PongGameWithPowerUps.py to play. Ensure Python 3 and pygame are installed.

Graphical Adventure

Download
Graphical Adventure is a visual text-based exploration game by Judah Johnson. Players navigate a strange environment using on-screen prompts and logic-based choices. The game blends storytelling and interactive puzzles with basic Python graphics. Ensure that Python 3 is installed on your device. Run the .py file in the unzipped folder to play!
